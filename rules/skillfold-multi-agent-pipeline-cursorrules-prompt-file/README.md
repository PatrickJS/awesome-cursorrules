# Skillfold Multi-Agent Pipeline

Cursor rules for multi-agent AI pipeline development, generated by [Skillfold](https://github.com/byronxlg/skillfold).

Skillfold is a configuration language and compiler for multi-agent AI pipelines. It compiles a single YAML config into `.cursor/rules/*.mdc` files and 10 other AI editor formats.

This `.cursorrules` file combines four skills (planning, code-writing, code-review, testing) into a dev-team pipeline with a planner, engineer, and reviewer agent.

## Usage

You can use this file directly, or generate custom pipeline rules with Skillfold:

```bash
npm install -g skillfold
skillfold init
skillfold --target cursor
```

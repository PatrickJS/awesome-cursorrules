# Skillfold (AI Pipeline Compiler)

Cursor rules for [Skillfold](https://github.com/byronxlg/skillfold) development.

Skillfold is a configuration language and compiler for multi-agent AI pipelines. It compiles YAML config into platform-specific agent skill files for Claude Code, Cursor, Windsurf, Codex, Copilot, Gemini, and more.

## Features

- Define reusable atomic skills as SKILL.md files
- Compose skills into agents via YAML configuration
- Wire agents into typed execution graphs with conditional routing, loops, and parallel map
- Compile to 12+ output targets (Claude Code, Cursor, Windsurf, Codex, Copilot, Gemini, Goose, Roo Code, Kiro, Junie, Agent Teams)
- Validate state schemas and team flows at compile time
- Run pipelines end-to-end with state management

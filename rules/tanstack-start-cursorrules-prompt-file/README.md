# TanStack Start Cursor Rules

Cursor rules for TanStack Start — the full-stack React framework built on TanStack Router and Vinxi. Covers server functions, API routes, streaming, SSR, authentication, and multi-platform deployment.

## What's covered
- Project structure and `app.config.ts` setup
- Root route HTML shell configuration
- `createServerFn` with Zod validation
- Using server functions in loaders and mutation handlers
- Raw API routes with `createAPIFileRoute`
- Streaming with `defer()` and Suspense
- Auth via server-side cookie/session access
- Deployment targets (Vercel, Netlify, Cloudflare, Bun, Node)
- TanStack Query integration for cache-first data loading

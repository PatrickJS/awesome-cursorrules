# Vibe Coding Directories .cursorrules

## Description
These rules empower Cursor AI to build and maintain directories focused on "vibe coding"—curated resources like rules, prompts (platform-customizable), MCP, news, jobs, and more. Perfect for creating fun, engaging coding hubs.

## Why Use This?
- Automates vibe-infused curation: Generate tailored prompts/rules while keeping it chill and creative.
- Example: See it in action at [Vibe Coding Directory](https://lovabledirectory.site/), a vibey directory with manually added Cursor resources and more.

## Installation
1. Copy `rules.yaml` to your project's root (or as .cursorrules if preferred).
2. In Cursor AI: "@cursor generate vibe prompt for Cursor platform."
